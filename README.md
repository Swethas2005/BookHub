# BookHub 

### 
"Discover Your Next Favorite Read: Welcome to Our Book Hub!"


## Tech stack used:
  - React.js
  - Chakra UI
  - React Router Dom
  - Axios
  - OpenLibrary API (for Data)


## Functionality
  - Routing (Navigating between the pages)
  - Data Fetching from the OpenLibrary API
  - Private Routing
  - Fully Responsive for all screens
